* Description
This layer adds support for editing .vue files.

** Features:
  - Syntax highlighting

* Install
To use this configuration layer, add it to your =~/.spacemacs=. You will need to
add =vue= to the existing =dotspacemacs-configuration-layers= list in this
file.
